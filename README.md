 ✈️ Travel Booking App UI

## 📖 Project Overview
The TripPink Travel Booking App UI is a mobile application interface designed using Figma. The application provides users with a simple and attractive interface to search for travel destinations, book flights, and manage their travel plans.
This project was created as part of the UI/UX Design Laboratory to demonstrate the principles of user interface design and user experience.

## 🎯 Objective
  To design an intuitive and visually appealing mobile UI for a travel booking platform. 
  To apply UI/UX design principles such as consistency, hierarchy, and usability in a real-world scenario. 
  To create a seamless user journey from destination search to flight booking and payment.---

## 🎨 Design Highlights

 Consistent pink-and-white color theme across all screens 
  Clean, card-based layout for destinations, flights, and pricing 
  Rounded corners and soft shadows for a modern, friendly look 
  Clear visual hierarchy (headings, prices, and CTAs stand out) 
  Simple, minimal navigation with icon-based bottom bar 
  Reusable UI components (cards, buttons, payment template) 
  Transparent fare breakdown for user trust 
  Consistent typography and spacing for readability
